Shared File Systems API v1 (SUPPORTED)

Provides coordinated access to shared or distributed file systems.

The Shared File Systems API v1 is functionally identical to the Shared File Systems API v2.0. Subsequent API v2 micro-versions, such as v2.1, differ from API v1.0.

Starting with API v2.0, the API uses Compute-style micro-versions.
